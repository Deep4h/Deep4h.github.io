Task and Challenges

Save/Load System

I managed the Saving System of Project. The challenge here was to create a system who take useful data in the Project and save them.

Gameplay Programmer

For the Gameplay I managed moving platforms, shooting trap, checkpoint system and all camera behaviours. The main challenge here was camera with its behaviours, they had to fit perfectly each other and the main behaviour is a very particular feature to create.

Tool Programmer

For the workflow I created a tool useful for managing scenes, for example you can add/remove easly scenes to/from build, change the scenes order into build and you can switch between them. The challenge here was to study the Engine Editor and Scenes for finding the correct way to do the tool.
