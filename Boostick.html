Task and Challenges

UI Programmer

I've created and managed all the UI. The challenge here was to set all UI Elements for a responsive UI that chenges based on Display Resolution. Another challenge was the implementation of achievement system and shop.

Save/Load System

I managed the Saving System of Project. The challenge here was to create a system who take useful data in the Project and save them.

Optimization

The main challenge was to obtain a build that work with a good framerate because the hardware had low specs.
